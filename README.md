# Grouvex Studios - DDOO (Documentos Oficiales)

Sistema de documentación oficial tipo BOE para Grouvex Studios con estructura jerárquica de organismos.

## Estructura de Organismos

- **Grouvex Studios** (Principal)
  - B.O.Grouvex Studios
  - Miembros de Grouvex Studios
  - N.O.Grouvex Studios
  - ToS&PP - Comunidad

- **Grouvex Studios Records**
  - B.O.GS-Records
  - Miembros de GS-Records
  - N.O.GS-Records
  - ToS&PP - GSRecords

- **Grouvex Studios Designs**
  - B.O.GS-Designs
  - Miembros de GS-Designs
  - N.O.GS-Designs
  - ToS&PP - Designs

- **Grouvex Studios Games**
  - B.O.GS-Games
  - Miembros de GS-Games
  - N.O.GS-Games
  - ToS&PP - Games

- **DDOO - Grouvex Studios** (Índice completo)

## Características

- ✅ Autenticación Firebase (solo @grouvex.com)
- ✅ Editor JSON con versionado local
- ✅ Historial de cambios en localStorage
- ✅ Filtros por organismo, tipo y año
- ✅ Diseño profesional responsive
- ✅ Exportación/importación de datos

## Instalación

1. Clona este repositorio
2. Configura Firebase Auth en `firebase-config.js`
3. Sube a GitHub Pages

## Uso

- **Página pública**: Visualización de documentos con filtros
- **Panel admin** (`/config.html`): Edición del JSON con versionado
